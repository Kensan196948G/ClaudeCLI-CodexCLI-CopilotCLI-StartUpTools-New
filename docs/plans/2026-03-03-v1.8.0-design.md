## Track 1: LogManager.psm1（新規モジュール）

### ファイル

`scripts/lib/LogManager.psm1`

### エクスポート関数

| 関数 | 概要 |
|------|------|
| `Start-SessionLog` | セッションログの開始（Start-Transcript ラッパー） |
| `Stop-SessionLog` | セッションログの停止 + SUCCESS/FAILURE サフィックス付与 |
| `Invoke-LogRotation` | keepDays に基づくログファイル削除 |
| `Invoke-LogArchive` | archiveAfterDays に基づくログファイルの ZIP 圧縮 |
| `Get-LogSummary` | ログ統計情報の表示（ファイル数、サイズ、最古ログ等） |

### ログファイル名形式

```
{prefix}-{project}-{browser}-{port}-{timestamp}.log
```

- **prefix**: config.json `logging.logPrefix`（デフォルト: `claude-devtools`）
- **project**: 選択されたプロジェクト名
- **browser**: `edge` / `chrome`
- **port**: DevTools ポート番号
- **timestamp**: `yyyyMMdd-HHmmss` 形式

**例**: `claude-devtools-MyProject-edge-9222-20260303-143022.log`

セッション終了時に結果サフィックスを付与:
- 成功時: `claude-devtools-MyProject-edge-9222-20260303-143022-SUCCESS.log`
- 失敗時: `claude-devtools-MyProject-edge-9222-20260303-143022-FAILURE.log`

### ログディレクトリ

- **優先**: config.json `logging.logDir`（スクリプトルートからの相対パス）
- **フォールバック**: `$env:TEMP`（logDir にアクセス不可の場合）
- ディレクトリ不在時は自動作成

### config.json `logging` セクション（既存）

```json
{
  "logging": {
    "enabled": true,
    "logDir": "logs",
    "logPrefix": "claude-devtools",
    "successKeepDays": 30,
    "failureKeepDays": 90,
    "archiveAfterDays": 30,
    "legacyKeepDays": 7
  }
}
```

### ログローテーションルール

| 対象 | 保持期間 | 動作 |
|------|---------|------|
| SUCCESS ログ | `successKeepDays` (30日) | 削除 |
| FAILURE ログ | `failureKeepDays` (90日) | 削除 |
| レガシーログ（サフィックスなし） | `legacyKeepDays` (7日) | 削除 |
| アーカイブ対象 | `archiveAfterDays` (30日) | ZIP 圧縮後に元ファイル削除 |

### Claude-DevTools.ps1 への統合

現在のハードコーディング:
```powershell
# Before (Claude-DevTools.ps1 lines 60-76)
$LogDir = $env:TEMP
$LogPrefix = "claude-devtools"
$LogPath = Join-Path $LogDir "${LogPrefix}-${LogTimestamp}.log"
Start-Transcript -Path $LogPath -Append -ErrorAction Stop
```

統合後:
```powershell
# After
Import-Module "$PSScriptRoot/../lib/LogManager.psm1"
$logResult = Start-SessionLog -Config $Config -ProjectName $ProjectName -Browser $Browser -Port $Port
# ... セッション実行 ...
Stop-SessionLog -Success $true  # または $false
Invoke-LogRotation -Config $Config
```

## Track 2: Linux 側ログ管理 + ScriptGenerator 統合

### ScriptGenerator.psm1 変更

`New-RunClaudeScript` 関数が生成する `run-claude.sh` に以下のログセクションを追加:

#### ログディレクトリ構造（Linux 側）

```
${PROJECT_ROOT}/
├── logs/
│   ├── claude-session-20260303-143022.log    # セッションログ
│   ├── mcp/
│   │   └── mcp-health-20260303-143022.log    # MCP ヘルスチェックログ
│   └── archive/
│       └── 2026-02.zip                        # 月次アーカイブ
```

#### run-claude.sh ログセクション

```bash
# --- ログ設定 ---
LOG_DIR="${PROJECT_ROOT}/logs"
MCP_LOG_DIR="${LOG_DIR}/mcp"
LOG_TIMESTAMP=$(date +%Y%m%d-%H%M%S)
SESSION_LOG="${LOG_DIR}/claude-session-${LOG_TIMESTAMP}.log"

mkdir -p "$LOG_DIR" "$MCP_LOG_DIR"

# stdout + stderr をファイルとターミナル両方に出力
exec > >(tee -a "$SESSION_LOG") 2>&1

# --- ログローテーション (30日超を削除) ---
find "$LOG_DIR" -maxdepth 1 -name "claude-session-*.log" -mtime +30 -delete 2>/dev/null
find "$MCP_LOG_DIR" -name "mcp-health-*.log" -mtime +30 -delete 2>/dev/null

# --- 月次アーカイブ ---
ARCHIVE_DIR="${LOG_DIR}/archive"
PREV_MONTH=$(date -d "last month" +%Y-%m 2>/dev/null || date -v-1m +%Y-%m 2>/dev/null)
if [ -n "$PREV_MONTH" ]; then
    ARCHIVE_FILES=$(find "$LOG_DIR" -maxdepth 1 -name "claude-session-${PREV_MONTH}*.log" 2>/dev/null)
    if [ -n "$ARCHIVE_FILES" ]; then
        mkdir -p "$ARCHIVE_DIR"
        zip -j "${ARCHIVE_DIR}/${PREV_MONTH}.zip" $ARCHIVE_FILES 2>/dev/null && rm -f $ARCHIVE_FILES
    fi
fi
```

#### MCP ヘルスチェックログ

既存の `mcp-health-monitor.sh`（tmux ペイン）の出力を `logs/mcp/` にも書き出す:

```bash
# mcp-health-monitor.sh 変更
MCP_LOG="${PROJECT_ROOT}/logs/mcp/mcp-health-$(date +%Y%m%d-%H%M%S).log"
# チェック結果をファイルにも記録
echo "[$(date)] MCP status: $STATUS" >> "$MCP_LOG"
```

#### 検証項目

| 検証 | 詳細 |
|------|------|
| 必須サーバー存在確認 | `.mcp.json` に `requiredServers` の各サーバーが定義されているか |
| トークン形式検証 | `githubToken` が `ghp_` / `github_pat_` プレフィックスか |
| トークンマスク表示 | ログ/UI表示時は `ghp_****abcd` 形式にマスク |
| 任意サーバー通知 | `optionalServers` 未設定時は警告（エラーではない） |

#### run-claude.sh への MCP チェック追加

```bash
# --- MCP サーバー検証 ---
MCP_JSON="${PROJECT_ROOT}/.mcp.json"
REQUIRED_SERVERS=("brave-search" "context7" "github" "memory" "playwright" "puppeteer" "sequential-thinking")

if [ -f "$MCP_JSON" ]; then
    echo "📋 MCP サーバー検証..."
    for server in "${REQUIRED_SERVERS[@]}"; do
        if grep -q "\"$server\"" "$MCP_JSON"; then
            echo "  ✅ $server"
        else
            echo "  ⚠️ $server (未設定)"
        fi
    done
else
    echo "⚠️ .mcp.json が見つかりません"
fi
```
