# GitHub Copilot CLI ベストプラクティス

- Copilot CLI は [scripts/main/Start-CopilotCLI.ps1](/D:/ClaudeCLI-CodexCLI-CopilotCLI-StartUpTools-New/scripts/main/Start-CopilotCLI.ps1) 経由で起動し、直接 `copilot --yolo` を叩く運用を標準にしない
- `.github/copilot-instructions.md` はテンプレート配備を基準にし、各プロジェクトで大きく逸脱させない
- 自動実行では `-NonInteractive` を付け、検証では `-DryRun` を使う
- SSH 実行を使う場合は `gh` がリモート側にも導入されていることを事前確認する
- 設定変更は `config/config.json.template` に先に反映し、必要な差分だけを `config/config.json` に持つ
