# GitHub Copilot CLI 概要

このリポジトリでは、GitHub Copilot CLI を [scripts/main/Start-CopilotCLI.ps1](/D:/ClaudeCLI-CodexCLI-CopilotCLI-StartUpTools-New/scripts/main/Start-CopilotCLI.ps1) から起動します。

現行仕様では Copilot CLI の基準コマンドを `copilot --yolo` に統一しています。

運用方針は次のとおりです。

- ローカル実行と SSH リモート実行の両方に対応する
- 設定は `tools.copilot.command` と `tools.copilot.args` に集約する
- プロジェクト起動時に `.github/copilot-instructions.md` を必要に応じて配備する
- 実行方法は Claude / Codex と同じ操作感に寄せる
- 自律開発の翻訳は custom agents + fleet を前提にする

共通思想は [11_自律開発コア.md](/D:/ClaudeCLI-CodexCLI-CopilotCLI-StartUpTools-New/docs/common/11_自律開発コア.md) を参照してください。
