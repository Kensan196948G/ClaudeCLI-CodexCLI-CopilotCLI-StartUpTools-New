# GitHub Copilot CLI 使い方

## 基本起動

### ローカル実行

```powershell
.\scripts\main\Start-CopilotCLI.ps1 -Project MyProject -Local
```

### SSH リモート実行

```powershell
.\scripts\main\Start-CopilotCLI.ps1 -Project MyProject
```

### 非対話モード

```powershell
.\scripts\main\Start-CopilotCLI.ps1 -Project MyProject -Local -NonInteractive
```

### 実行確認のみ

```powershell
.\scripts\main\Start-CopilotCLI.ps1 -Project MyProject -Local -DryRun
```

## 起動時の動作

起動スクリプトは次の処理を行います。

1. `config/config.json` を読み込む
2. `gh` コマンドが利用可能か確認する
3. 作業対象プロジェクトを決定する
4. `.github/copilot-instructions.md` を必要に応じて配備する
5. `tools.copilot.env` を反映する
6. `copilot --yolo` を起動する

## 補足

現行実装では `copilot --yolo` を前提にしています。ドキュメント、設定ファイル、起動スクリプトはこの基準に統一されています。
