# Claude Code 使い方ガイド

> 対象ツール: Claude Code（Anthropic）
> ドキュメント種別: 操作方法・起動フロー

---

## 基本的な起動方法

このリポジトリでの Claude Code 利用は、`scripts/main/Start-ClaudeCode.ps1` を中心に行います。現行構成では旧 DevTools/tmux 前提ではなく、ローカル起動または SSH 起動を選ぶ方式です。

### 方法1: `start.bat` から起動

```cmd
start.bat
```

メニューで次を選択します。

- `S1` SSH 経由で Claude Code を起動
- `L1` ローカルで Claude Code を起動

### 方法2: PowerShell から直接起動

```powershell
.\scripts\main\Start-ClaudeCode.ps1
.\scripts\main\Start-ClaudeCode.ps1 -Project "my-project"
.\scripts\main\Start-ClaudeCode.ps1 -Project "my-project" -Local
.\scripts\main\Start-ClaudeCode.ps1 -Project "my-project" -NonInteractive
.\scripts\main\Start-ClaudeCode.ps1 -Project "my-project" -DryRun
```

## 起動フロー

### ローカル起動

```text
1. config.json を読み込む
2. claude コマンドの存在確認
3. projectsDir 配下からプロジェクト選択
4. CLAUDE.md テンプレートを必要に応じて配備
5. 環境変数を設定して Claude Code を起動
```

### SSH 起動

```text
1. config.json を読み込む
2. claude コマンドの存在確認
3. sshProjectsDir 配下からプロジェクト選択
4. bash スクリプトを生成して base64 で転送
5. linuxHost 上の linuxBase/{project} で Claude Code を起動
```

---

## プロジェクト選択

`-Project` を省略すると、設定に応じて次の場所から対話選択されます。

- ローカル起動: `projectsDir`
- SSH 起動: `sshProjectsDir`

`-NonInteractive` では `-Project` 指定が必須です。

```powershell
.\scripts\main\Start-ClaudeCode.ps1 -Project "backend-api" -NonInteractive
```

---

## 設定ファイルとの関係

Claude Code 起動に使う主な設定:

| キー | 用途 |
|------|------|
| `projectsDir` | ローカル起動時のプロジェクトルート |
| `sshProjectsDir` | SSH 起動時の Windows 側共有ドライブ |
| `linuxHost` | SSH 接続先 |
| `linuxBase` | Linux 側プロジェクトルート |
| `tools.claude.args` | Claude 起動引数 |
| `tools.claude.env` | Claude 起動時の環境変数 |

例:

```json
{
  "linuxHost": "192.168.0.185",
  "linuxBase": "/home/kensan/Projects",
  "tools": {
    "claude": {
      "args": ["--dangerously-skip-permissions"],
      "env": {
        "ENABLE_TOOL_SEARCH": "true",
        "CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS": "1"
      }
    }
  }
}
```

## 認証

Claude Code は次のどちらかで利用します。

- `Anthropic Max / Pro` などのサブスクリプション認証
- `ANTHROPIC_API_KEY` による API 認証

サブスクリプション利用時:

```powershell
claude
# 起動後に /login
```

API 利用時:

```powershell
$env:ANTHROPIC_API_KEY = "sk-ant-..."
```

## テンプレート配備

起動スクリプトは必要に応じて、対象プロジェクトに `CLAUDE.md` を配備します。テンプレートは [scripts/templates/CLAUDE.md](/D:/ClaudeCLI-CodexCLI-CopilotCLI-StartUpTools-New/scripts/templates/CLAUDE.md) にあります。

ローカル起動時:
- プロジェクト直下に `CLAUDE.md` を配置

SSH 起動時:
- Linux 側のプロジェクト直下に `CLAUDE.md` を転送

## よく使う起動例

```powershell
# 対話的に SSH 起動
.\scripts\main\Start-ClaudeCode.ps1

# ローカル起動
.\scripts\main\Start-ClaudeCode.ps1 -Project "my-app" -Local

# 非対話で SSH 起動
.\scripts\main\Start-ClaudeCode.ps1 -Project "my-app" -NonInteractive

# 実行内容だけ確認
.\scripts\main\Start-ClaudeCode.ps1 -Project "my-app" -DryRun
```

## Claude Code 内での基本操作

```text
/help        ヘルプ表示
/clear       会話履歴をクリア
/compact     コンテキスト圧縮
/status      現在の状態確認
/model       モデル切り替え
/login       認証
```

詳細なコマンド仕様は Claude Code 本体の最新ヘルプを参照してください。
