# Claude Code セットアップガイド

> 対象ツール: Claude Code（Anthropic）
> ドキュメント種別: インストール・初期設定

---

## 前提条件

| 要件 | 内容 |
|------|------|
| Node.js | 18 以上 |
| npm | 利用可能であること |
| OS | Windows / macOS / Linux |
| 認証 | サブスクリプション認証 または API キー |

確認:

```powershell
node --version
npm --version
```

## インストール

```powershell
npm install -g @anthropic-ai/claude-code
claude --version
```

アップデート:

```powershell
npm update -g @anthropic-ai/claude-code
```

## 認証

### サブスクリプション認証

```powershell
claude
# 起動後に /login
```

### API 認証


```powershell
[System.Environment]::SetEnvironmentVariable(
  "ANTHROPIC_API_KEY",
  "sk-ant-...",
  "User"

```

## このリポジトリでの設定

Claude Code は `config/config.json` の `tools.claude` セクションで管理します。

例:

```json
{
  "tools": {
    "claude": {
      "enabled": true,
      "command": "claude",
      "args": ["--dangerously-skip-permissions"],
      "env": {
        "ENABLE_TOOL_SEARCH": "true",
        "CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS": "1",
        "CLAUDE_AUTOCOMPACT_PCT_OVERRIDE": "50",
        "CLAUDE_CODE_ENABLE_PROMPT_SUGGESTION": "true"
      },
      "settings": {
        "language": "日本語",
        "outputStyle": "Explanatory",
        "alwaysThinkingEnabled": true
      }
    }
  }
}
```

主に使うキー:

| キー | 用途 |
|------|------|
| `projectsDir` | ローカル起動時のプロジェクトルート |
| `sshProjectsDir` | SSH 起動時の Windows 側共有ドライブ |
| `linuxHost` | SSH 接続先 |
| `linuxBase` | Linux 側プロジェクトルート |
| `tools.claude.args` | Claude 起動引数 |
| `tools.claude.env` | 起動時に設定する環境変数 |

## 起動確認

```powershell
.\scripts\main\Start-ClaudeCode.ps1 -DryRun
```

または:

```cmd
start.bat
```

ローカル起動確認:

```powershell
.\scripts\main\Start-ClaudeCode.ps1 -Project "my-project" -Local
```

SSH 起動確認:

```powershell
.\scripts\main\Start-ClaudeCode.ps1 -Project "my-project"
```

---

## Linux 側の前提

SSH で Claude Code を起動する場合は、Linux 側に次が必要です。

- `claude`
- `bash`
- `base64`
- `linuxBase` 配下の対象プロジェクト

必要に応じて Linux 側でも Claude Code をインストールしてください。

```bash
npm install -g @anthropic-ai/claude-code
claude --version
```

---

## 補足

- 現行構成では `~/.claude/settings.json` への自動マージを前提にしていません。
- 実際の起動制御は `config/config.json` と `Start-ClaudeCode.ps1` を基準にしてください。
- 旧構成の DevTools / tmux / `run-claude.sh` 前提は現行標準ではありません。
