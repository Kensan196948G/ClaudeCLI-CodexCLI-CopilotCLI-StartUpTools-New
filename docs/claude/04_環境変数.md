# Claude Code 環境変数リファレンス

> 対象ツール: Claude Code（Anthropic）
> ドキュメント種別: 環境変数リファレンス

---

## 概要

このリポジトリで Claude Code に渡す環境変数は、主に `config/config.json` の `tools.claude.env` に定義します。ローカル起動時は PowerShell プロセスへ設定され、SSH 起動時はリモート bash スクリプトの `export` として設定されます。

例:

```json
{
  "tools": {
    "claude": {
      "env": {
        "ENABLE_TOOL_SEARCH": "true",
        "CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS": "1",
        "CLAUDE_AUTOCOMPACT_PCT_OVERRIDE": "50",
        "CLAUDE_CODE_ENABLE_PROMPT_SUGGESTION": "true"
      }
    }
  }
}
```

---

## 主に使う環境変数

### `ANTHROPIC_API_KEY`

| 項目 | 内容 |
|------|------|
| 用途 | Anthropic API 認証 |
| 必須 | API 利用時のみ |
| 形式 | `sk-ant-` で始まる文字列 |

```powershell
$env:ANTHROPIC_API_KEY = "sk-ant-..."
```

サブスクリプション認証を使う場合は必須ではありません。

### `CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS`

| 項目 | 内容 |
|------|------|
| 用途 | Agent Teams の有効化 |
| 推奨値 | `"1"` |

```json
"CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS": "1"
```

### `ENABLE_TOOL_SEARCH`

| 項目 | 内容 |
|------|------|
| 用途 | ツール検索機能の有効化 |
| 推奨値 | `"true"` |

### `CLAUDE_AUTOCOMPACT_PCT_OVERRIDE`

| 項目 | 内容 |
|------|------|
| 用途 | 自動コンパクトのしきい値 |
| 推奨値 | `"50"` |

### `CLAUDE_CODE_ENABLE_PROMPT_SUGGESTION`

| 項目 | 内容 |
|------|------|
| 用途 | プロンプトサジェストの有効化 |
| 推奨値 | `"true"` |

---

## 設定場所の優先順

このリポジトリで運用する場合は、通常次の順で考えれば十分です。

1. 実行中シェルで直接設定した値
2. `Start-ClaudeCode.ps1` が `tools.claude.env` から設定した値
3. Claude Code 側の既定値

---

## `config.json` での設定例

```json
{
  "tools": {
    "claude": {
      "env": {
        "ENABLE_TOOL_SEARCH": "true",
        "CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS": "1",
        "CLAUDE_AUTOCOMPACT_PCT_OVERRIDE": "50",
        "CLAUDE_CODE_ENABLE_PROMPT_SUGGESTION": "true"
      },
      "settings": {
        "language": "日本語",
        "outputStyle": "Explanatory",
        "alwaysThinkingEnabled": true,
        "spinnerTipsEnabled": true,
        "prefersReducedMotion": false,
        "promptSuggestionEnabled": true,
        "respectGitignore": true,
        "autoUpdatesChannel": "latest",
        "includeCoAuthoredBy": true
      }
    }
  }
}
```

---

## 現行構成で扱わないもの

現行標準フローでは、次の旧構成前提の変数は前提にしていません。

- `CLAUDE_CHROME_DEBUG_PORT`
- `MCP_CHROME_DEBUG_PORT`
- DevTools ポート番号に依存する設定
- `run-claude.sh` 前提の変数適用

必要であれば個別プロジェクトで追加運用できますが、このリポジトリの標準仕様ではありません。

---

## セキュリティ上の注意

- API キーをリポジトリにコミットしない
- `config.json` に秘密情報を直接保存しない
- 永続化する場合はユーザー環境変数を使う
