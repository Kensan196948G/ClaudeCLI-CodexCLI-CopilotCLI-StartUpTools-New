
```powershell
pwsh -NoProfile -File .\scripts\test\Test-McpRuntime.ps1
pwsh -NoProfile -File .\scripts\test\Test-McpRuntime.ps1 -OutputFormat Json
```

## SSH 接続できない

```powershell
ssh your-linux-host "echo OK"
ssh -vvv your-linux-host
```

- `linuxHost` が正しい
- SSH 鍵が配置されている
- Linux ホストに到達できる

## Cron HTML メールレポート (v3.2.0)

| 症状 | 原因 | 対処 |
|---|---|---|
| `EOF: command not found` | heredoc の終端 `EOF` が行頭インデント付きで認識されず、`EOF`/`chmod` 等の文字列がファイルに混入 | 1) `rm -f ~/.env-claudeos` で削除 → 2) `echo 'export ...' >> ~/.env-claudeos` を 1 行ずつ実行する方式で再作成 |
| `WARN: CLAUDEOS_SMTP_USER ... 未設定` | cron 環境に環境変数が渡っていない(`~/.bashrc` は cron で読まれない) | `cron-launcher.sh` 冒頭の `source ~/.env-claudeos` を確認(`grep -n 'env-claudeos' ~/.claudeos/cron-launcher.sh`)。重複追加されている場合は `sed -i '<重複行番号>d'` で削除 |
| `(535, b'5.7.8 Username and Password not accepted')` | アプリパスワードが間違い / 2 段階認証が無効 | https://myaccount.google.com/apppasswords で再発行。スペース付き 19 文字 / 除去 16 文字どちらでも可(両形式を Gmail SMTP は受付) |
| `scp ... :~/.claudeos/: Failure` | 宛先ディレクトリ未作成 | `ssh kensan@host 'mkdir -p ~/.claudeos && chmod 700 ~/.claudeos'` を先に実行 |
| `ssh: Could not resolve hostname <your-linux-host>` | プレースホルダをそのままコピペ | 実ホスト名(例: `kensan1969`)に置換するか SSH config の別名を使用 |
| メール届かない・ログにも何も出ない | `CLAUDEOS_EMAIL_ENABLED=1` 未設定(既定 off) | `~/.env-claudeos` の最終行に `export CLAUDEOS_EMAIL_ENABLED=1` を追加 |
| `command not found: python3` | Python 3 未導入 | `sudo apt install python3` 等(標準ライブラリのみ使用するため pip 不要) |
| `cron-launcher.sh` の source 行が 2 重 | sed -i '/^pattern/a ...' を冪等性なしで 2 回実行 | `grep -n 'env-claudeos' ~/.claudeos/cron-launcher.sh` で重複確認 → `sed -i '<行番号>d'` で削除 |

詳細手順は [`docs/common/16_HTMLメールレポート設定.md`](./16_HTMLメールレポート設定.md) を参照。

## それでも解決しない場合

以下を添えて調査してください。

- Windows バージョン
- PowerShell バージョン
- 実行したスクリプト名
- エラーメッセージ全文
- `config/config.json` の機密情報を除いた内容
- Cron 関連の場合: `~/.claudeos/logs/cron-*.log` の末尾 50 行 + `crontab -l | grep CLAUDEOS`
