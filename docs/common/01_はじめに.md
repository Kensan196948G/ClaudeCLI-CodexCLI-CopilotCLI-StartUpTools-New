# はじめに

このドキュメント群は、`Claude Code`、`Codex CLI`、`GitHub Copilot CLI` を Windows から統一的に起動するための現行ガイドです。

---

## このプロジェクトでできること

- `start.bat` からツールを選んで起動
- Windows ローカルで各ツールを起動
- SSH 経由で Linux 上の `claude` / `codex` / `copilot --yolo` を起動
- 設定を `config/config.json` に集約
- 診断スクリプトで導入状態を確認

---

## 対応ツール

### Claude Code

- 複数ファイルをまたぐ実装・レビュー向き
- 本プロジェクトではローカル起動と SSH 起動の両方に対応

### Codex CLI

- ターミナル作業やコード生成を素早く進めたい場合に向く
- `AGENTS.md` を対象プロジェクトへ配備可能

### GitHub Copilot CLI

- `copilot --yolo` による対話型コーディング支援
- `.github/copilot-instructions.md` を対象プロジェクトへ配備可能
