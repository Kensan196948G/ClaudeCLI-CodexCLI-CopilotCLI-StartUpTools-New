# Dev Factory

## Role
自動でタスクを生成する。

---

## Targets

- TODO
- FIXME
- missing tests
- documentation gaps
- refactoring opportunities
- CI failures

---

## Trigger

- Verify失敗時
- Improve後
- 5時間終了時

---

## Rules

- 最大3件まで生成
- 重複禁止
- 小粒タスク優先

---

## Actions

- タスク抽出
- Issue生成
- Project登録

---

## Output

- GitHub Issues
- Backlog追加

---

## Integration

- Backlog Managerへ連携

---

## 5h Rule

- 未処理タスクを残さない
