# ClaudeOS v8 セッション開始

## セッション開始処理

自律実行は Linux cron（月〜土、プロジェクト別スケジュール・300分制限）が担う。
セッション開始時に loop や schedule コマンドの登録は不要。

Codex が利用可能な場合のみ実行する。**Codex が使えなくても自律開発は止めない。**

```
/codex:setup
/codex:status
```

- Codex 認証済み → review 強化レイヤーとして活用
- Codex 未認証 / 使えない → このステップをスキップして自律開発を開始する
- **Codex の有無は停止条件ではない**
