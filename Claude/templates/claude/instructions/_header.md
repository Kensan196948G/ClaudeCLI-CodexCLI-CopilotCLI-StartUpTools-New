## セッション開始処理

自律実行は Linux cron（月〜土、プロジェクト別スケジュール・300分制限）が担う。
セッション開始時に `/loop` や `/schedule` の登録は不要。
直ちに Codex セットアップを実行してから自律開発を開始すること。

```
/codex:setup
/codex:status
```
