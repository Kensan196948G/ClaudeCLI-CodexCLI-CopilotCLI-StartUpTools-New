# Token Budget Manager

Weekly token management.

## Budget Zones

| Zone   | Range    | Status                  |
| ------ | -------- | ----------------------- |
| Green  | 0–60%    | Normal development      |
| Yellow | 60–75%   | Reduced build activity  |
| Orange | 75–90%   | Monitor priority        |
| Red    | 90–100%  | Development stopped     |
