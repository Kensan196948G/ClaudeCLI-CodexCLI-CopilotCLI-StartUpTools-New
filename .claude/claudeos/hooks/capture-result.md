# capture-result

PostToolUse の通常フックとして起動するツール結果変換器。
主要ツールの実行結果を、後続の判定・記録処理で扱いやすい形式に変換する。

本ファイルは静的なドキュメントではなく、**呼び出された Claude への命令書** である。
Claude Code の hook システムがこのフックを起動したとき、以下の実行契約を順に処理し、
完了してから戻ること。

---

## 位置づけ

| 項目 | 値 |
|---|---|
| フック種別 | PostToolUse |
| 対象ツール | 全ツール（matcher: `.*`） |
| 副作用 | ツール結果の正規化・構造化 |

---

## 実行契約

1. 直前のツール実行結果を確認する
2. 結果が `usage-history-recorder` や `onboarding-refresh-on-stable` で
   利用可能な形式かを確認する
3. 必要であれば結果を正規化し、後続フックが参照できる状態にする
4. 正規化が不要な場合は何もしない（ノーオペレーション）

---

## 関連フック

- `usage-history-recorder` — ツール使用履歴の記録
- `onboarding-refresh-on-stable` — STABLE 到達時のオンボーディング更新
